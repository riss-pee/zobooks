


# 📚 DigiBuks – Mizoram’s Digital eBook Ecosystem

## 1. Project Overview

**DigiBuks** is a **Mizoram-centric digital eBook platform** that enables users to **buy, rent, and read books digitally**, while empowering **local authors and publishers** to onboard, publish, and monetize their work seamlessly.

Inspired by global platforms like **Amazon Kindle**, DigiBuks is tailored specifically to **Mizo language, culture, and literary heritage**, creating a unified digital marketplace for regional literature.

The platform will be:

* **Intuitive & fun to use**
* **Visually aesthetic and professional**
* **Accessible to both tech-savvy and non-technical users**
* **Optimized for mobile-first experiences**

---

## 2. Vision & Mission

### Vision

To become the **primary digital publishing and reading platform for Mizoram**, preserving local literature while making it globally accessible.

### Mission

* Digitize and promote **Mizo authors and publishers**
* Encourage **reading culture** through affordable eBooks and rentals
* Provide a **fair and transparent monetization model** for authors
* Build a **sustainable digital literary ecosystem**

## 7. Technology Stack

### 📱 Frontend (Mobile App)

* **Flutter** – Cross-platform mobile development
* **GetX** – State management, routing, dependency injection
* **Dio** – API communication
* **Flutter Secure Storage** – Token & key storage

---

### 🧠 Backend (Suggested)

* Django / Django REST Framework
* PostgreSQL (structured data)
* Redis (caching & sessions)
* Celery (background tasks)
* MinIO / AWS S3 (eBook storage)

---

### 🔐 Authentication & Security

* JWT (Access & Refresh Tokens)
* Role-based access control (Reader / Author / Admin)
* Secure payment verification

---

### 💳 Payment Integration

* Razorpay / UPI / Cards
* Wallet support (future)
* Automated royalty payouts
